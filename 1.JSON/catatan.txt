Di PHP
function json_encode() = digunakan untuk mengubah array atau object yang kita punya menjadi json
function json_decode() = digunakan untuk mengubah json menjadi array
function file_get_contents = salah satu cara untuk mengambil json

Di Javascript
JSON.stringify() = function json_encode()
JSON.parse() = function json_decode()
